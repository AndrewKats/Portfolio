-------------------------------------
REQUIRED FEATURES AND HOW TO USE THEM
-------------------------------------
1. Registration: On the login (index) page, click "Register a new account" and provide information

2. Visit: To add a visit to your cart, start by using the search function to find a POI and look at it's information.
On the POI's page is a link "Add a Visit". Provide information to add a visit to this POI. To confirm (or remove) 
these visits, go to the "Confirm added visits" page either from the visitAdded page or the homepage.
	
3. New POI: While logged in as an admin, go into Admin options and choose "Add a new POI". Provide the information.

4. Update POI: While logged in as an admin, go into Admin options and choose "Update the info of a POI". Provide the information.

5. Favorites: Navigate to a POI using the search function then choose "Add to Favorites".

6. Feedback: Navigate to a POI using the search function then choose "Give Feedback".

7. Rate Feedback: Navigate to a POI using the search function and all its feedback will be shown. 
Choose a usefulness option below the feedback to rate it.

8. Trust: From the homepage choose "Trust/Distrust a user". Enter a login and choose whether to trust or distrust that user.

9. POI Browsing: From the homepage choose "Search for points of interest". Fill in any information to filter results then
press submit. Leave all the fields blank if you just want to see all POI's. The POI's will be displayed with their ID's.
Enter an ID and press submit to go to the page of that POI.

10. Useful Feedbacks: Navigate to a POI page then fill in the Limit box below "See most useful feedbacks" and press Submit.

11. Visit Suggestions: Navigate to a POI and add a visit. Once added, suggestions will be given.

12. Degrees of Separation: From the homepage choose "Find degrees of separation between two users". Enter two logins and submit.

13. Statistics: From the homepage choose "See category statistics" then choose which stat to see. Enter a category and limit.

14. User Awards: From the homepage choose "Admin options" then choose either "Find most trusted users" or "Find most useful users."
Enter a limit and submit to see results.
